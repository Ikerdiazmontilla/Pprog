# Pprog

Second iteration of the conversational game project for PPROG.

## Build

Use:

```sh
make
```

The project is configured to compile with `-Wall -ansi -pedantic` and without warnings.

## Run

Use:

```sh
make run
```

or:

```sh
./castle castle.dat
```

## Tests

Build every test executable with:

```sh
make tests
```

Run the available unit tests with:

```sh
./set_test
./space_test
./character_test
```
