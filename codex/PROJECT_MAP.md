# PROJECT_MAP

- Repository root: C modules for the PPROG adventure game (`game`, `space`, `player`, `object`, `character`, `set`, command loop and graphic engine).
- `other_files/`: assignment PDFs, style guide, scratchpad, delivery artifacts and Gantt spreadsheets.
- `codex/`: persistent assistant memory (`LESSONS.md`, `RULES.md`, `PROJECT_MAP.md`) plus temporary working files under `codex/tmp/`.
- `castle.dat`: canonical playable dataset used by `make run`.
